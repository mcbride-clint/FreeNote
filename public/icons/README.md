# App Icons

Drop your PWA icons here to make MarkFlow installable with a proper app icon:

- `icon-192.png` — 192×192 PNG
- `icon-512.png` — 512×512 PNG
- `icon-512-maskable.png` — 512×512 PNG with safe area for maskable icons

The PWA manifest references these paths (see `vite.config.ts`).

Until you add them, the app is still fully functional — only the installed-app icon will use a browser default.
